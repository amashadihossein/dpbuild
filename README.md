product.

## Installation

For released version

``` r
remotes::install_git(url = "https://github.com/amashadihossein/dpbuild.git")
```

For dev version

``` r
remotes::install_git(url = "https://github.com/amashadihossein/dpbuild.git",
                     ref = "dev")
```
